Diseño Web

El diseño web es aquella actividad consistente en planificar, diseñar, mantener y crear páginas web.
Se aleja del término más tradicional del diseño en cuanto a que engloba una gran variedad de aspectos diferentes, como pueden ser el diseño de la interfaz, el diseño del material gráfico o incluso la experiencia del usuario como el sitio.

Diseño grafico

Se dedican a crear diseños creativos para comunicar mensajes o ideas de manera única que atraiga la atención de quien los vea

Desarrollo web

El Desarrollo Web es una de las actividades imprescindibles dentro de la dinámica digital actual ya que es a través de él y del uso de diversas tecnológicas, que se pueden realizan las páginas web.
Con el fin de que los usuario de internet, puedan obtener una adecuada experiencia digital.

¿Quienes Somos?

WolvesSupreme es un proyecto personal y me gusta la idea de brindar un sitio web para dar informacion sobre lanzamientos de animes, videojuegos y peliculas.
No soy un experto solamente soy un persona como ustedes con ganas de compartir información y La verdad es que no saco ningún beneficio, tan solo la satisfacción de haber ayudo a muchas personas.
Ya que nosotros pasamos por el mismo proceso de no conocer sitios web donde nos brinde informacion suficiente por lo cual decidimos crear este pequeño proyecto que facilite a todas las personas que visiten este sitio web
